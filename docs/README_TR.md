[Bruce Firmware](https://github.com/pr3y/Bruce) üzerinde özel bir **Arsenal** modülü ile yapılmış bir değişiklik. Orijinal Bruce özelliklerinin tamamı korunmuştur.

## Kurulum

1. `.bin` dosyasını [Releases](../../releases) adresinden indirin
2. [ESP Web Flasher](https://espressif.github.io/esptool-js/) (Chrome/Edge) ile veya Arsenal Dashboard üzerinden OTA ile flaşlayın
